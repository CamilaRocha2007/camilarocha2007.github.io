Mas o que é HTML?

HTML (HyperText Markup Language) é uma linguagem de marcação usada para criar páginas da web. Ele estrutura o conteúdo em elementos como títulos, parágrafos, imagens e links, e é interpretado por navegadores para exibir o conteúdo de forma organizada.
